臺北市100年度提昇教師自由軟體應用能力培訓計畫

壹、目的

由本市高中職以下學校主動提出自由軟體應用3～6小時研習計畫，經臺北市自由軟體桌面應用推廣小組評選8所學校後，開放本市有興趣教師參與。不但可激勵各校對自由軟體導入教學進行研究，同時也可收見賢思齊之效。

貳、辦理單位

一、主辦單位：臺北市政府教育局

二、承辦單位：臺北市新湖國民小學

參、研習時間：100年3月1日~7月30日

肆、研習地點：各研習協辦學校

伍、研習課程

與自由軟體在教學、行政應用有關之主題。

陸、參加對象

臺北市高中職、國中小等學校教職員。

柒、報名方式

即日起至研習日期前三日，逕上臺北市電子研習護照報名，每場次錄取人數依研習地點預計人數為準。

捌、研習時數

全程參與教師依實核予研習時數。

玖、獎勵

本計畫相關承辦人員於計畫執行後，由教育局從優辦理敘獎。

壹拾、經費來源

由教育局相關經費項下支應。

壹拾壹、本計畫陳  核後實施，修正時亦同。
