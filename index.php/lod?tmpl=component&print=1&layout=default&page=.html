臺北市100年度主題隨身碟暨LOD系統維護人員研習計劃

壹、目的

ㄧ、為宣導智慧財產權，減少非法軟體之使用，推廣自由軟體與自由教材， 促進校園軟體與教學多元發展。

二、整合分類線上可運用於教學情境自由軟體與自由教材，並製作成可攜式以方便教師課堂上可以直接運用教學。

三、培養可以定期更新LOD系統上的自由軟體與教材以及中文化，便利本市教師取得較新版本的軟體或教材，增進課堂教學應用。

貳、辦理單位

 

ㄧ、主辦單位：臺北市政府教育局

二、承辦單位：臺北市華江高級中學

肆、研習對象

自由軟體推動小組成員參加。

伍、報名方式

即日起至研習前三日，逕上臺北市電子研習護照報名，每場次錄取人數 ，依研習地點預計人數為準。

陸、研習時數

一、全程參與教師核予十八小時。

二、請假在四小時(含)以內者，依實核予研習時數。

三、請假超過四小時以上者，依實核予研習時數。

 

柒、獎勵

本計畫相關承辦人員於計畫執行後，由教育局從優辦理續獎。

捌、經費來源

由教育局相關經費項下支應。
