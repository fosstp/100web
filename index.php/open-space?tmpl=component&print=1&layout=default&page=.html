以開放原始源碼為應用核心的open space教學創意計畫

壹、前言

目前本市已積極建置未來教室及實施電子書包運用，然而其所需的成本頗高，以致難以在各級科任教室實施。開放源碼不但成本低廉，且能針對教學現場需求提供適合的教學資源，再加上社群的努力，不少開放源碼都已完成中文化，即使是國小學生亦可輕易上手。再結合以wii接受器建置軟體式電子白板、本小組研發的主題式隨身碟、moodle學習平台及行動輔具，將可在最短時間，以最低的成本，建置open space，並善用教師教學創意，讓師生教學更具效能。

貳、目的

一、建置高度互動及合作學習之未來學習環境。

二、培養學生具備創新、批判性思考、問題解決及有效溝通與合作等能力。

三、發展無所不在的學習，以擴大學習機會。

四、提昇教師的資訊素養及專業知能，充分運用資訊化社會中的數位工具與資源進行教學，並激勵學生的學習和創造力。

參、辦理單位

一、主辦單位：臺北市政府教育局

二、承辦單位：臺北市新湖國民小學

肆、工作內容及執行方式

一、高互動教學情境建置－軟體式電子白板及moodle平台

實體式電子白板價格偏高，且移動不便。以wii接受器結合、電腦藍芽

接受器，及學校的單槍設備，即可快速建置軟體式電子白板，並在室內

外室內、外使用，高互動環境即可在教學場域中實施。另外，建置moodle

平台，不但方便教師運用軟體式電子白板教學，同時也可助於學生進行

合作學習。

二、U化學習環境建置－主題式隨身碟及行動輔具

小組所製作的主題式隨身碟內含無線廣播、心智圖、瀏覽器、電子白板

軟體、及PhET等教學資源，只需將此隨身碟利用USB界面與電腦、行

動輔具連接，該裝置便立刻變身為電子書包，讓師生都能藉此進行行動

學習。

伍、階段推廣計畫

本計畫以100年度以2所自由軟體推廣學校為種子示範學校，未來計畫以每年提供本市10所中小學提出申請導入，十年共100所中小學分年度推廣，若超過10所則必須於隔年才能再提出申請，相關經費預算也會在隔年再申請編列。
