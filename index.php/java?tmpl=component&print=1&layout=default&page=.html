臺北市100年度Java手機程式設計融入資訊教育研習計畫

壹、目的

有鑑於智慧型手機日漸普及，手機上的應用也愈來愈廣泛，而Java程式語言跨平台的特性，更是跨平台應用程式開發語言首選。培訓本市中小學教師運用自由軟體開發手機應用程式，進一步協助指導學生應用自由軟體於生活中的資訊產品，並於資訊課程中倡導自由軟體的應用與推廣。

貳、辦理單位

一、主辦單位：臺北市政府教育局

二、承辦單位：臺北市立華江高級中學

參、研習時間：100年8月3日、4日、5日（共三天）

肆、研習地點：臺北市立華江高級中學

伍、研習課程

課程 時數 日期 研習地點 預計人數

Java手機程式設計融入資訊教育 18小時 100年8月3、4、5日(三天) 華江高中 40人

陸、參加對象

臺北市高中職、國中小等學校資訊組長、資訊教師及一般老師有興趣者。

柒、報名方式

即日起至研習日期前三日，逕上臺北市電子研習護照報名，每場次錄取人數依研習地點預計人數為準。

捌、研習時數

一、全程參與教師核予十八小時。

二、請假在四小時(含)以內者，依實核予研習時數。

三、請假超過四小時以上者，依實核予研習時數。

玖、獎勵

本計畫相關承辦人員於計畫執行後，由教育局從優辦理敘獎。

壹拾、經費來源

由教育局相關經費項下支應。

壹拾壹、本計畫陳  核後實施，修正時亦同。
